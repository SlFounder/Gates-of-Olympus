# Gates-of-Olympus
Gates of Olympus — популярный слот от Pragmatic Play с каскадами, множителями до x500 и фриспинами, где выигрыш может расти цепочками. Игра отличается высокой волатильностью и огромным потенциалом заноса. Боги Олимпа, особенно Зевс, могут активировать множители в любой момент, превращая обычный спин в эпическую победу.
